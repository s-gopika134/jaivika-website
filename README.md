# Jaivika – Style for Every You

Jaivika is a modern fashion website designed for men and women clothing collections.

## Features
- Custom designed logo with heart-shaped V
- Clean and elegant UI
- Responsive layout
- Men & Women product sections
- Add to cart button (frontend functionality)
- Stylish hero section

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Type
Frontend Web Development Project

This project is built as part of my web development learning journey and portfolio.
